## README

This mod unlocks "The World's Game" for offline play

## HOW TO INSTALL

1. Unzip "UNLOCK_WORLD_CUP.zip"
2. Move the `data` folder to `C:\FC 26 Live Editor\mods\legacy`

![](https://i.imgur.com/ny881pw.png)

3. Verify installation. Check, if you have the `eventMainHubFlow.nav` file in `C:\FC 26 Live Editor\mods\legacy\data\ui\nav`. If yes then the you have installed the mod properly.

![](https://i.imgur.com/NKhzTlF.png)
